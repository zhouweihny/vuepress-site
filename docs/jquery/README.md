# jQuery 教程

> 基于《锋利的jQuery》一书，系统学习 jQuery 的知识体系。

## 概述

jQuery 是一个快速、简洁的 JavaScript 库，它简化了 HTML 文档遍历、事件处理、动画和 AJAX 交互。自 2006 年发布以来，jQuery 影响了整个前端开发领域。

## 核心内容

### 选择器
jQuery 的选择器几乎继承了所有 CSS 选择器语法，并且扩展了许多实用的过滤选择器。

[阅读选择器详情 →](./选择器.md)

### DOM 操作
用简洁的语法完成元素的增删改查，告别浏览器兼容性问题。

[阅读 DOM 操作 →](./dom操作.md)

### 事件处理
统一的事件绑定机制，支持事件委托，解决浏览器差异。

[阅读事件处理 →](./事件.md)

### 动画效果
内置 fade、slide、animate 等动画方法，轻松实现视觉效果。

[阅读动画效果 →](./动画.md)

### AJAX 请求
简化异步请求，支持 GET、POST 及完整的配置选项。

[阅读 AJAX →](./ajax.md)
